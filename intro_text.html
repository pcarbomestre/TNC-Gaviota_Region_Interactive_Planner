Marine Institute Shellfish Fisheries Database

An interactive website for the display of Shellfish fisheries data in Ireland.

 WHAT YOU CAN DO ON THIS WEBSITE:

  • Visualize scientific data collected around the Irish coastline as part of the MI Observer and Sentinel Vessel Fleet Programme (SVP) (Sampling Programmes Tab).
  • Check the latest updated figures on Irish landings for Shellfish species (Landings Tab). 
  • Consult Stock Assessment outputs and Management Advice for several Shellfish stocks (Assessment and Advice Tab). 

This website is in continuous development. More data sources, assessments and relevant information of Shellfish Fisheries in Ireland will be added over time.

DISCLAIMER: the data displayed in this website is inspected for potential errors that can arise for different reasons (e.g data entry mistakes). However, some issues might escape our data checks. Similarly, some stocks or areas, have little data support. Users must therefore exercise caution if extracting conclusions from certain plots.
